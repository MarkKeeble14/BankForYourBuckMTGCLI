# MTG 'Bang for your Buck' CLI

This is a CLI built to analyze which sets and precons to purchase if you're trying to find a specific set of cards

## To use

#### Step 1: Clone the repo
```bash
git clone https://github.com/MarkKeeble14/MTGPackToPurchase.git
```

#### Step 2: Enter the cloned directory
```bash
cd MTGPackToPurchase
```

#### Step 3: Open the .sln file
```bash
start MTGSetRecommender.sln
```

#### Step 4: Build the project (Can be done by pressing Shift+F6)

#### Step 5: Back in your command line, execute the run script
```bash
run
```

#### Step 6: Play!

## External Libraries

The following external libraries are used in this project:
- Curl: [RESTful API Calls](https://github.com/curl/curl.git)
- JSON Parsing: [json (nlohmann)](https://github.com/nlohmann/json.git)
