# MTG 'Bang for your Buck' CLI

This is a CLI built to analyze which set boosters and precons to purchase if you're trying to find a specific set of cards

## To use

#### From Command Line

### Step 1: Clone the repo to your local machine from the command line
```bash
git clone https://github.com/MarkKeeble14/MTGPackToPurchase.git
```

#### Step 2: Enter the cloned directory
```bash
cd BankForYourBuckMTGCLI
```

#### Step 3: Run the program
```bash
run
```

#### From Browser 

### Step 1: Download the files from the [repo](https://github.com/MarkKeeble14/BankForYourBuckMTGCLI) in your browser

### Step 2: Unzip the downloaded files in your file explorer

### Step 3: Run the program by doubleclicking run.bat

#### File Input
The input file is located at the root of the project, named cards_list.txt

The format within should be:
quantity card name

# For Example:
1 Sol Ring
1 Satoru, the Infiltrator
1 Arcane Signet
1 Soul Diviner
1 Fellwar Stone

## External Libraries and Resources

The following external libraries are used in this project:
- Curl: [RESTful API Calls](https://github.com/curl/curl.git)
- JSON Parsing: [json (nlohmann)](https://github.com/nlohmann/json.git)

And the precon data is coming from the following repo:
- CommanderPrecons: [Precon Data](https://github.com/Westly/CommanderPrecons/tree/500b65cb20941fb0cc26a3c5149088d45f8bbe9b)
